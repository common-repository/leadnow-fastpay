== Description ==

Leadnow.pl - skuteczna sieć afiliacyjna

Jesteś właścicielem serwisu internetowego? Masz stałe grono zaangażowanych odbiorców? Wykorzystaj to! Dołącz do sprawdzonej sieci afiliacyjnej i otrzymuj wynagrodzenie za przeprowadzone kampanie reklamowe. Masz do dyspozycji kilkadziesiąt programów.

Wtyczka wspomagająca integrację leadnow.pl z Wordpressem

Plugin posiada trzy funkcje:
* Możliwość łatwego osadzenia widgeta Fastpay.pl
* Wstawianie linków reklamowych Leadnow.pl w wygodnej api, dzięki której nie będą blokowane przez Adblocka.
* Możliwość łatwego osadzenia iframe do programu video.
